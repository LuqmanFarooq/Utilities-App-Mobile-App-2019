Instructions on App Usage

The home Tab is Events Search Engine where you can search events happening at specific city which is defined by user by inputing city name E.g galway

Important! Make Sure you Spell the city name Correctly

The News Tab is where you Get latest headlines with images and there is a button on the top left corner for taking Screenshots

The location page uses geolocation plugin and storage plugin to show current location and last location by saving it

The about page holds all the information of the app.

You can click on github button to go to developes github repository

The contact page where holds all the contact info of the developer

Clicking on Email Address launches the default mail app to send email to that address.

Get started now!
